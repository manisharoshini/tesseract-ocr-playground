# OCR with PyTesseract and OpenCV

This repository contains a Python script demonstrating various Optical Character Recognition (OCR) techniques using **PyTesseract** and **OpenCV**.  
It includes examples for text extraction, language-specific OCR, bounding box visualization, exporting results, and more.

- OCR with timeout  
- Character bounding boxes  
- Detailed OCR data
